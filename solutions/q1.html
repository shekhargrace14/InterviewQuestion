Q1. Write a function that returns the reverse of a string?

                        console.log(reverseString("I am a Software Engineer"))

                        There are two ways to solve these types of questions.
                        1. with the help of loops  and conditions.
                        2. With the help of JavaScripts in build functions.
                        
                        // 1st approach.
                        So now lets solve this question with 1st approach. 
                        // using for loop 
                        function reverseString(str){
                            // Initialize an empty string to store the reverse string
                            let reversed = ""
                            
                                //Iterate through the characters of the input string in reverse order
                                
                                for(let i = str.length-1; i>1 >=0; i--){
                                    reversed +=str[i];

                                }
                                return reversed;
                        }
